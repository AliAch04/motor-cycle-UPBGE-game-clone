# Traffic Rider Clone
A motorcycle traffic racing game built with UPBGE 0.5

## Requirements
- UPBGE 0.5 → https://github.com/UPBGE/upbge/releases

## How to Play
1. Open traffic_rider.blend in UPBGE 0.5
2. Hover mouse over 3D Viewport
3. Press P to start

## Controls
| Key | Action |
|-----|--------|
| W | Accelerate |
| S | Brake |
| A | Steer Left |
| D | Steer Right |
| ESC | Quit |
